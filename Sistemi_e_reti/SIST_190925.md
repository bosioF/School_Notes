# OS / Linux

## Formattazioni dei dischi

- **NTFS** → usato da Windows
    
- **ext4** → usato da Linux
    

**Partizioni di sistema:**

- Windows parte da `C:`
    
- Linux parte da `/` (**root**)
    

**Cartelle principali in Linux:**

|Directory|Contenuto|
|---|---|
|`/bin`|eseguibili di sistema|
|`/home`|directory degli utenti (`/home/studente`, `/home/docente`, ecc.)|

---

## Interfacce

- **GUI** → Graphical User Interface (interfaccia grafica)
    
- **CLI** → Command Line Interface (interfaccia testuale)
    

---

## Sistema operativo

Il sistema operativo gestisce la comunicazione tra **hardware** e **software**.

---

# Linux

Linux è un **sistema operativo libero e open source**, basato su **API POSIX** e su **UNIX**.

**Componenti principali:**

- **Kernel** → nucleo del sistema, gestisce processi, driver e memoria centrale; fa da intermediario tra software e hardware
    
- **Shell** → interprete dei comandi (es. `bash` in `/bin/sh`)
    
- **Applicativi** → software utilizzabile dall’utente
    

---

### Prompt della shell

Esempio:

```
nome@computer:~$
```

Significato:

- `nome` → utente connesso
    
- `computer` → hostname della macchina
    
- `~` → home dell’utente (`/home/$USER`)
    
- `$` → utente normale (`#` indica root)
    

> Nota: il terminale Linux è **case sensitive** (distingue maiuscole e minuscole)

---

## Comandi di base

|Comando|Funzione|
|---|---|
|`ls`|lista il contenuto della directory|
|`cd`|cambia directory|
|`pwd`|mostra il percorso della directory corrente|
|`;`|concatena più comandi sulla stessa riga|
|`mkdir`|crea una nuova directory|
|`rmdir`|elimina una directory vuota|
