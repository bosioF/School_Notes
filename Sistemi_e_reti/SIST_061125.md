# Bash Scripting

## Operatori matematici nei test (`[ ]` o `[[ ]]`)

| Operatore | Significato                | Esempio             |
| --------- | -------------------------- | ------------------- |
| `-lt`     | minore di (`<`)            | `[ "$a" -lt "$b" ]` |
| `-le`     | minore o uguale a (`<=`)   | `[ "$a" -le "$b" ]` |
| `-gt`     | maggiore di (`>`)          | `[ "$a" -gt "$b" ]` |
| `-ge`     | maggiore o uguale a (`>=`) | `[ "$a" -ge "$b" ]` |
| `-eq`     | uguale a (`==`)            | `[ "$a" -eq "$b" ]` |
| `-ne`     | diverso da (`!=`)          | `[ "$a" -ne "$b" ]` |

---

### Esempio di utilizzo in uno script Bash

```bash
#!/bin/bash

a=5
b=10

if [ "$a" -lt "$b" ]; then
    echo "$a è minore di $b"
else
    echo "$a non è minore di $b"
fi
```
