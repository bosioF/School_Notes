# Bash – Vettori

## Dichiarazione di vettori

### Metodo 1 – con tutti gli elementi in una volta

```bash
arr_0=(a b c d e f)
```

### Metodo 2 – elemento per elemento

```bash
arr_1[0]="a"
arr_1[1]="b"
arr_1[2]="c"
arr_1[3]="d"
```

### Metodo 3 – copiare un vettore in un altro

```bash
arr_2=(${arr_1[*]}) # tutti gli elementi di arr_1 vengono copiati in arr_2
```

### Vettore vuoto

```bash
empty_arr_0=()
declare -a empty_arr_1

empty_arr_0[0]="a"
empty_arr_1[0]="a"
```

---

### Stampare tutti gli elementi

```bash
echo "${arr_2[*]}" # OUTPUT: a b c d
```

---

## Accesso agli elementi

- Per accedere all’elemento con indice `i` di un vettore `arr` si usa:
    

```bash
${arr[i]}
```

> Nota: fuori dalle parentesi tonde si usa `${arr[i]}`, dentro parentesi tonde per operazioni si può usare anche `arr[i]`.

---

## Contare elementi di un vettore

```bash
arr=(a b c d e f)
arr_len=${#arr[*]}
echo "$arr_len"  # OUTPUT: 6
```

---

## Contare la lunghezza di un elemento

```bash
arr=(a bb ccc dddd eeeee ffffff)
elem_len=${#arr[3]}  # lunghezza dell'elemento "dddd" = 4
echo "$elem_len"      # OUTPUT: 4
```
