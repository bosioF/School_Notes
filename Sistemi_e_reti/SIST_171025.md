# Bash Scripting

Ogni script in **bash** inizia con lo **shebang**, che nel nostro caso è:

```bash
#!/bin/bash
```

### Comandi di Base

#### `mv` - Spostare o rinominare file e cartelle

- **Sintassi**: `mv <origine> <destinazione>`
    
    - **Spostare una cartella**:
        
        ```bash
        mv <cartella> <cartella_destinazione>
        ```
        
        In questo caso, `<cartella>` viene spostato nella cartella `<cartella_destinazione>`.
        
    - **Rinominare una cartella**:
        
        ```bash
        mv <cartella> <nuovo_nome>
        ```
        
        In questo caso, la cartella `<cartella>` viene rinominata in `<nuovo_nome>`.
        

#### `cp` - Copiare file e cartelle

- **Sintassi**: `cp <origine> <destinazione>`
    
    - **Copiare una cartella**:
        
        ```bash
        cp -r <cartella> <cartella_destinazione>
        ```
        
        In questo caso, `<cartella>` viene copiato nella cartella `<cartella_destinazione>`. L'opzione `-r` (ricorsiva) è necessaria per copiare intere cartelle.
        
    - **Copiare e rinominare una cartella**:
        
        ```bash
        cp -r <cartella> <cartella_destinazione>/<nuovo_nome>
        ```
        
        In questo caso, `<cartella>` viene copiato in `<cartella_destinazione>` e rinominato come `<nuovo_nome>`.
