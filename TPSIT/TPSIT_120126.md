# JavaScript

## Introduzione

JavaScript è un **linguaggio di programmazione ad eventi**, utilizzato per rendere le pagine web dinamiche.

---

## Funzioni

Sintassi di una funzione freccia:

```javascript
const func = () => {
    // codice da eseguire
}
```

### Esempio di integrazione con HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>funzione esempio</title>
</head>
<body>
    <input type="text" id="inputText" placeholder="Scrivi qualcosa">
    <input type="button" value="Cliccami" onclick="checkInput()">

    <script>
        const checkInput = () => {
            const input = document.getElementById("inputText");

            // controlla se il campo è vuoto
            if (input.value.length === 0) {
                alert("Inserisci del testo!");
            } else {
                alert("Hai scritto: " + input.value);
                // cambia lo stile del campo
                input.style.backgroundColor = "lightyellow";
            }
        }
    </script>
</body>
</html>
```

## Selezionare oggetti nel DOM

- `document.querySelector(selector)` → seleziona il **primo elemento** che corrisponde al selettore CSS
    
- `document.querySelectorAll(selector)` → seleziona **tutti gli elementi** che corrispondono al selettore
    
- `document.getElementById(id)` → seleziona un elemento per `id` (più lenta rispetto a `querySelector`)
    

---

## Controlli condizionali (`if`, `else`)

```javascript
const text = document.getElementById("inputText");

if (text.value.length > 0) {
    console.log("Hai scritto qualcosa!");
} else {
    console.log("Campo vuoto!");
}
```

---

## Modifica di stile e contenuto

- Cambiare lo **stile CSS** di un elemento:
    

```javascript
text.style.backgroundColor = "lightblue";
text.style.fontSize = "16px";
```

- Modificare il **testo interno**:
    

```javascript
text.value = "Nuovo testo"; // oppure .innerHTML per usare HTML, .innerText per il testo
```

- Controllare la **lunghezza del testo**:
    

```javascript 
console.log(text.textLength);
```

oppure

```javascript
console.log(text.value.length);
```
