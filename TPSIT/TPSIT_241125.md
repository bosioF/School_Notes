## Bootstrap

Bootstrap si importa tramite CDN inserendo questo tag nel `<head>` della pagina HTML:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css"        rel="stylesheet"        integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC"        crossorigin="anonymous">
```

Bootstrap utilizza un **sistema a griglia (grid system)**:

- Una **riga (`row`)** può contenere **al massimo 12 colonne**
    
- Le colonne si definiscono con classi come `col-1` … `col-12`
    
- La somma delle colonne in una riga **non deve superare 12**
