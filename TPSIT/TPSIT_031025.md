## Appunti HTML e CSS

## HTML

### Tag `<div>`

#### A cosa serve?

Il tag `<div>` serve per creare un **contenitore rettangolare** generico.

#### Cosa si può inserire in un `div`?

All’interno di un `div` si può inserire **qualsiasi contenuto**:

- testo
    
- immagini
    
- video
    
- altri `div`
    

Caratteristiche importanti:

- Il `div` occupa **il 100% della larghezza** del contenitore padre
    
- L’**altezza si adatta al contenuto**
    
- Ogni `div` ha un **bordo**, anche se di default **non è visibile**
    

---

## CSS

### A cosa serve?

Il CSS serve per **modificare l’aspetto grafico** dei tag HTML (colori, dimensioni, posizione, ecc.).

Può essere inserito:

- direttamente nell’HTML con il tag `<style></style>`
    
- oppure tramite file esterno `.css`
