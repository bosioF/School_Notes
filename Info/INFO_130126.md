# Ripasso sulle Funzioni in C

In un programma C ci sono **tre aree principali**:

1. **Area dei prototipi** → dichiarazioni delle funzioni
    
2. **Area del `main`** → corpo principale del programma
    
3. **Area delle funzioni** → definizione delle funzioni
    

---

### Esempio completo

```c
// Area dei prototipi
int func(int, int);

int main() { // Area del main
    int x = 5, y = 3;
    
    int z = func(x, y); // chiamata della funzione
    printf("La somma è: %d\n", z);
    return 0;
}

// Area delle funzioni
int func(int a, int b) {
    int sum = a + b;
    return sum;
}
```

---

### Concetti chiave

#### Chiamante

Il **chiamante** è il punto del programma dove viene eseguita la funzione:

```c
int z = func(x, y); // func(x, y) è il chiamante
```

#### Parametri attuali

I **parametri attuali** (o argomenti) sono i valori passati alla funzione:

```c
int z = func(x, y); // x e y sono parametri attuali
```

#### Parametri formali

I **parametri formali** sono le variabili definite nella funzione che ricevono i valori dei parametri attuali:

```c
int func(int a, int b) { ... } // a e b sono parametri formali
```
