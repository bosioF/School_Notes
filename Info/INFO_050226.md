#C
## Puntatori

Un **puntatore** è una variabile che contiene come valore **l’indirizzo di memoria** di un’altra variabile.

### Esempio base

```c
int var = 10;
int *ptr = &var;

printf("%d, %p", var, ptr);

// OUTPUT: 10, 0x...
```

- `&var` → restituisce l’indirizzo di memoria di `var`
    
- `ptr` → contiene l’indirizzo di `var`
    
- `%p` → specificatore corretto per stampare indirizzi
    

---

## Dichiarare un puntatore

La sintassi generale è:

```c
tipo *nome_puntatore;
```

Esempio:

```c
int var = 10;
int *ptr = &var;   // puntatore a int che punta a var
```

Altri esempi:

```c
float *fptr;
char *cptr;
```

> Il tipo del puntatore **deve corrispondere** al tipo della variabile puntata.

---

## Perché usare i puntatori?

I puntatori permettono di:

- Modificare direttamente una variabile
    
- Evitare copie inutili di dati
    
- Passare grandi strutture alle funzioni in modo efficiente
    
- Lavorare con array, stringhe e memoria dinamica
    

### Esempio: passaggio per riferimento

```c
#include <stdio.h>

void add10(int *num){
    *num += 10;
}

int main(){
    int a = 10;
    add10(&a);
    printf("%d", a);
}
```

**Output**

```
20
```

Senza puntatori, `a` verrebbe copiato e il valore originale non cambierebbe.

---

## Dereferenziazione di un puntatore

L’operatore `*` permette di **accedere o modificare** il valore contenuto all’indirizzo puntato.

```c
int x = 10;
int *ptr = &x;

*ptr = 20;  // x diventa 20
```

- `ptr` → indirizzo di `x`
    
- `*ptr` → valore contenuto in `x`
    

---

## Copiare il valore di una variabile tramite puntatore

```c
int x = 10;
int *ptr = &x;
int y = *ptr;  // y = 10
```
